# 📂 Riicuy Script - Ultimate Collection

Selamat datang di repositori **Riicuy Script**, koleksi script otomatisasi dan antarmuka pengguna (UI) terbaik untuk pengalaman bermain game yang lebih lancar dan cerdas.

---

## 🚀 Fitur Utama

### 1. Riicuy Auto Sambung-Kata (v2.2.0)
Script otomatisasi untuk permainan sambung kata yang dilengkapi dengan AI pintar.
* **Auto-Answer:** Menjawab kata secara otomatis menggunakan database kata bahasa Indonesia.
* **Smart Delay:** Pengaturan jeda waktu (ms) agar terlihat seperti diketik oleh manusia dan menghindari deteksi.
* **Customization:** Atur panjang kata (Min/Max) dan tingkat agresivitas AI melalui menu.
* **Mobile Optimized:** Menggunakan Rayfield Library yang ringan dan stabil untuk perangkat mobile.

### 2. Riicuy UI Library
Kerangka kerja antarmuka pengguna profesional untuk membuat menu script Anda sendiri.
* **Multi-Theme:** Mendukung tema Dark, Purple, dan Red.
* **Animated UI:** Transisi dan efek animasi yang halus.
* **Key System:** Dilengkapi dengan sistem keamanan kunci (Key System).
* **Drag Support:** Bisa digeser di layar PC maupun perangkat Touch (Mobile).

### 3. Update Notification System
Sistem pemberitahuan otomatis untuk menginformasikan pengguna saat script sedang dalam pemeliharaan atau pembaruan.

---

## 🛠️ Cara Penggunaan

1. Salin kode dari file `.lua` yang ingin Anda gunakan (disarankan `sambung-kata-v2.lua` untuk fitur terbaik).
2. Buka *executor* pilihan Anda di perangkat Mobile atau PC.
3. Tempelkan kode tersebut dan tekan **Execute**.
4. Menu **Riicuy Script** akan muncul di layar Anda.
